# eth-network-exts

Network extension types for Ethereum and Layer 2 chains.

## Overview

This crate provides trait definitions and implementations for different Ethereum-based networks, allowing type-safe network configuration and compatibility with both Alloy and Reth node types.

## Supported Networks

- Ethereum Mainnet
- Base Mainnet
- Sepolia Testnet
- Unichain Mainnet

## Features

- Unified trait (`EthNetworkExt`) for network configuration
- Chain ID constants for each network
- Type-safe integration with Alloy and Reth
