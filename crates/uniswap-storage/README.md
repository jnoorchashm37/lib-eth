# uniswap-storage

Storage utilities for fetching and managing Uniswap protocol data.

## Overview

This crate provides functionality to fetch and store data from various Uniswap protocol versions, including V3, V4, and Angstrom. It supports integration with both remote providers and local Reth nodes.

## Features

- `v3` - Uniswap V3 protocol support
- `v4` - Uniswap V4 protocol support
- `angstrom` - Angstrom protocol support
- `revm` - REVM database integration
- `local-reth` - Local Reth node integration

## Supported Functionality

- Storage fetching utilities for on-chain data
- Integration with Alloy providers
- REVM database support for local execution
- Type definitions for Uniswap protocols
